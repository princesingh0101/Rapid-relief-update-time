# RapidRelief Admin Panel – User & Store Management System

## Objective

Upgrade the **User Management module** in the RapidRelief Admin Panel to make it fully functional with backend integration and professional UI.

The system should allow the admin to manage:

Users
Stores
User Roles
Account status

All actions must update the backend database automatically.

---

# 1. Sidebar Update

Add a new sidebar item under **Users**.

Sidebar structure:

```
Dashboard
Inventory
Orders
Users
Stores
Settings
Logout
```

The **Stores** item must include an icon.

Example icon:

```
Store / Shop Icon
```

---

# 2. Users Management Page

Route:

```
/admin/users
```

This page should display all registered users.

Each user card or row must contain:

```
User Avatar
User Name
Email
Role
Status
Actions
```

Example roles:

```
Customer
Store Owner
Admin
Delivery Partner
```

---

# 3. Add User Feature

Add button at the top right:

```
+ Add User
```

Clicking this button opens a modal form.

Fields:

```
Full Name
Email
Phone Number
Password
Role (Dropdown)
Status (Active / Suspended)
Profile Image Upload
```

After clicking **Create User**:

```
User saved in database
User list updates automatically
```

---

# 4. Edit User Feature

Each user must have an **Edit icon**.

```
✏ Edit
```

Clicking edit should:

1. Open edit modal
2. Pre-fill user data.

Editable fields:

```
Name
Email
Phone
Role
Status
Profile Image
```

After saving:

```
Database updates
User list refreshes
```

---

# 5. Delete User Feature

Each user row must have a delete button.

```
🗑 Delete
```

Clicking delete must show confirmation dialog:

```
Are you sure you want to delete this user?
```

Options:

```
Cancel
Delete
```

After delete:

```
User removed from database
UI updates instantly
```

---

# 6. Search Users

Add search bar at the top.

Search fields:

```
Name
Email
Phone
Role
```

Search results must update instantly.
